# NeuroMeme AI

Welcome to the **NeuroMeme AI** project! This is a modern, professional AI meme website that allows users to generate memes, view trending memes, and contact the owner. The website features a cyberpunk theme and is built using Next.js and TypeScript.

## Features

- **Meme Generation**: Users can create memes using an AI-powered generator.
- **Trending Memes**: View a collection of trending memes generated by users.
- **Contact Form**: Users can reach out to the owner with questions or feedback.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/neuro-meme-ai.git
   ```
2. Navigate to the project directory:
   ```
   cd neuro-meme-ai
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Running the Development Server

To start the development server, run:
```
npm run dev
```
Open your browser and go to `http://localhost:3000` to view the application.

## Deployment

To deploy the application on Netlify, follow these steps:

1. Create a new site on Netlify and link it to your GitHub repository.
2. Set the build command to `npm run build` and the publish directory to `out` or `build` depending on your setup.
3. Add environment variables in the Netlify dashboard as needed.
4. Enable the contact form by configuring it in the Netlify Forms settings.
5. Deploy the site and ensure everything is functioning correctly.
